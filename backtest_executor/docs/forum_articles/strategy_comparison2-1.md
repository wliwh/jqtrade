# ETF轮动策略研究(二)附录：长牛资产池中的测试

在前文的分析中我们可以看到，动量策略在包含纳指和黄金的基准池中表现卓越，其本质原因在于这两类资产具备良好的长牛属性。这引发了我们的进一步思考：**如果我们将资产池极致浓缩，仅仅保留黄金和纳指这两种长期斜向上的资产，不同复杂度的择时策略能否通过轮动创造超额收益？**

为此，我们补充了本项测试：继续使用上文探讨的 `wy03`（原版动量）、`long`（长短期结合）和 `yj15`（综合择时）三种策略；资产池严格限定为“黄金+纳指”；回测区间设定为2023年1月1日至2026年2月15日（该阶段内这两种资产均展现出明显的单向上行趋势）。

## 三、分析与结论

从上述在“黄金+纳指”双资产池中的测试数据中，我们可以直观地感受到“好池子”决定了策略的收益下限与上限。对比 `wy03` 在基准4只资产中的表现和其在仅有“黄金+纳指”资产池中的表现，剔除较弱资产后，动量策略的总收益从128.21%跃升至176.78%，最大回撤从28.54%大幅缩减至16.73%，夏普比率更是近乎翻倍。这再次印证了前文的核心观点：动量策略的优秀表现，本质上是池子中强势资产（如黄金、纳指的长牛走势）赋予的。

此外，在面对极具长牛特性的资产时，往往“少即是多（低频胜过高频）”。观察三种不同复杂度的配置策略，最简单的原版动量策略 `wy03` 反而表现最好。它是唯一跑赢了“等权无脑持有”（基准收益152.09%）的策略，创造了24.68%的实际超额收益。相比之下，加入了更复杂的综合择时和止损逻辑的 `long` 与 `yj15` 策略，不仅未能增加超额收益，反而大幅跑输基准（超额分别为 -21.60% 和 -51.05%），成了反向指标。

其深层原因在于，复杂的择时极易导致“过度交易”与摩擦损耗。深挖交易频次指标可以发现，`yj15` 的换手率最高（0.18），交易次数高达140次，平均持仓天数极短；反观 `wy03` 换手率极低（0.03），一共只有26次交易。对于黄金和纳指这种长期呈斜坡向上趋势的组合，它们之间时不时会出现同涨同跌的共振。频繁切换常常导致策略无法吃满完整的上涨波段，反而因为短线噪音波动陷入“左右挨打”的窘境。例如，`wy03` 持仓按天数计算的命中率高达73.84%，说明它一旦咬住强势资产往往能握定；而 `yj15` 的天数命中率仅为44.91%，这意味着越是复杂的择时指标，在面对纯粹的长牛资产时，越容易因为过度敏感而频繁发出错误换仓信号。

综上所述，如果构建好的ETF篮子里装的已经是长坡厚雪的卓越资产，最有效的方法就是依靠基础的动量因子“持有最强，减少折腾”。复杂的止损、多重指标共振等风控逻辑，往往是为了应对充满震荡和系统性风险的市场（如过去的A股）；如果硬套在长牛资产（纳斯达克、黄金）为主的池子中，反而会因为过度防守和频繁换车，成为拖累长期业绩的最大累赘。
