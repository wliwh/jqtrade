# ETF轮动策略研究(三下)：重构寻优流程与资产池适配

## 参数寻优全流程重构

为了提升参数搜索的**科学性**、**易用性**与**分析深度**，我们在最近的迭代中对原有的评估流程进行了深度重构。重构后的框架将碎片化的流程标准化为三个阶段

1. 参数表生成过程，通过配置参数范围表和默认参数表，轻松实现参数表的生成
2. 调参过程，标准化为两个核心阶段：
    -   **阶段1：开关组合筛选**，通过对止损 (`ls`)、均线过滤及短期动量等功能模块进行全排列组合测试，识别出对策略稳定性贡献最大的核心功能组件，锁定最优的功能配置基准。
    -   **阶段2：核心参数精细化**，在确定功能开关后，针对评分上限 (`S`)、成交量过滤阈值 (`v`) 以及拟合优度 (`r`) 等关键连续参数进行全网格搜索，寻找收益与回撤的最佳平衡点。
3. 回测结果展示过程，将每个回测的参数配置显示在表格中，方便用户进行参数对比分析，还可显示逐年收益；优化了回测结果表格的输出样式

1. 在包含33只标的的ETF池子(`ETF33`)中，原版单纯动量策略由于极易被短期异动欺骗，收益仅为720%，夏普比率0.71；而在加入得分上限(`S`)、近期大跌保护(`ls`)和日内止损(`dl`)后，策略精确排除了“假趋势”，总收益发生了爆发式增长达到**4381%**，夏普比率飙升至1.91。同样，在包含海外与商品的中大池子 (`ETFIntF`, `ETFAwidIntF`) 中，优化版策略也取得了数倍于原版的超额收益。这证明这套多重过滤机制在管理多品种复杂池子时是核心“大杀器”。
2. 在仅含4只长牛标的的基准池 (`DEFAULT`) 中，由于池子本身极具含金量（纳指+黄金），优化版策略的收益并没有超过原版策略（甚至略微落后），因为止损机制可能会导致错失部分回调反弹的收益。但优化版显著压低了最大回撤（从 30.66% 降至 23.23%），夏普比率提升至1.27，依然起到了“平滑净值曲线”的保安作用。
3. A股真的会让人不幸，在纯A股宽基 (`ETFAwid`) 和包含A股行业的池子 (`ETFAall`) 中，不论是否优化，策略的夏普比率都在零甚至负值徘徊，收益极低且最大回撤巨大（高达60%左右）。这说明，A股高度相关且呈现强烈的脉冲式轮动，违背了动量策略“赢家恒赢”的基础前提。在没有长期趋势的泥潭里，过滤只会让人频繁“止损在低点”或“追高被套”。
4. 观察波动率、最大回撤、换手率数据可以发现，在所有取得池子中，优化方案的换手率往往高出原版两倍甚至更多，而波动率和最大回测均会下降。这说明优化的过滤机制极大地缩短了对极端行情的反应时间。在复杂市场中，策略以更高的交易频率和换仓成本为代价，换取了避开急跌的能力，确保了长期的极高复利。
