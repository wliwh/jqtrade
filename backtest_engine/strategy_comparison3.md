# ETF轮动策略研究(三上)：怎么过滤掉不合适的ETF

在本系列前两篇（[轮动池构建](https://www.joinquant.com/view/community/detail/d0cdab1fc98dec1cead8a4de29cc13a7)、[长牛资产池](https://www.joinquant.com/view/community/detail/bd01e11bcb836b543b5002f29a5b48e1)）研究中，我们深入探讨了标的池对绩效的决定性影响。然而，当我们将视角投向更广泛的市场时，单纯的动量追踪极易受到噪声干扰。在这种情况下，如何通过精细化的过滤手段剔除伪信号、规避系统性风险，成为了提升策略上限的关键。

本文讨论的轮动方案近期在社区中广为流传，是许多进阶策略（如[动量轮动](https://www.joinquant.com/view/community/detail/c1d9d9f97e86c845ad6be3318a95c4f8?type=1)、[高收益](https://www.joinquant.com/view/community/detail/35bd04e284dfe5523138c58002b410fb?type=1)）的核心原型。该策略在延续价格线性回归的基础上，主要在以下几个维度进行了深度改良：

1. 交易时间改到开盘一小时后，将当天交易前的价格拼接到历史价格序列中。并使用**加权线性回归**，赋予近期数据更高权重；
2. 极大增加了候选ETF池，扩展到33只，涵盖了国内宽基、行业ETF、债权、海外指数、商品指数等类别；
3. 添加了众多过滤措施，包括评分范围、近期大跌、均线、交易量、评分稳定性（R2）、短期收益等等过滤方法，具体条件和参数可见下面的表格；
4. 增设了日内止损机制，有效应对极端行情。

在保持策略核心逻辑不变的基础上，我对数据获取与底层架构进行了深度优化，显著提升了回测效率。

**风控与过滤参数详情**

| 过滤维度 | 代码参数名 | 状态 | 设定值 / 阈值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| **评分范围** | `SCORE_RANGE` | 开启 | 0.0 - 6.0 | 仅选择评分在此区间内的标的，避开过热标的 |
| **近期大跌保护** | `LOSE_PARAM` | 关闭 | 0.95 (-5%) | 近3日某日跌幅超过5%则触发拦截 |
| **成交量异常** | `VOLUME_PARAM` | 关闭 | 0.6倍 (5日均量) | 当日实时成交量超过5日均量的0.6倍则拦截(防止放量出货)，*注意：默认交易时间为10:31,10:32* |
| **均线过滤** | `MA_PARAM` | 关闭 | 20日均线 | 价格需在20日均线之上 |
| **R2 独立过滤** | `R2_PARAM` | 关闭 | 0.5 | 拟合优度 R^2 需大于 0.5 |
| **短期动量过滤** | `SHORT_MOMENTUM_PARAM` | 关闭 | 10,0.0 | 10日年化收益率需大于0 |
| **年化收益过滤** | `ANNUAL_RETURN_PARAM` | 关闭 | 1.0 (100%) | 回归年化收益率需大于100% |
| **日内止损** | `DAY_LIMIT_PARAM` | 开启 | 0.95 (-5%) | 相较于成本价亏损达5%时日内卖出 |

## 附

### 参数间相关性分析

| 参数对 | 相关性 | 原因 |
| :--- | :--- | :--- |
| `S` ↔ `r` | **强相关** | S 控制评分宽度，r 控制评分稳定性，两者联合决定标的筛选质量 |
| `S` ↔ `v` | 弱相关 | 成交量过滤独立于评分维度 |
| `v` ↔ `r` | 无关 | 分别作用于不同维度（量/价） |

### 搜索步骤

**子空间 A：联合搜索 S × r（核心参数对）**

由于 `S` 和 `r` 强相关，必须联合搜索。固定 `v` 为默认关闭，`dl`开启：

- S 候选值：4, 5, 6, 7, 8, 10（6个）
- r 候选值：off, 0.4, 0.5, 0.6, 0.7（5个）
- 组合数：**6 × 5 = 30 组**

从结果中选出 Calmar Ratio（总收益/最大回撤）最优的 S\* 和 r\*。

**子空间 B：单独搜索 v（成交量过滤）**

固定 S=S\*, r=r\*，搜索最优的成交量阈值：

- v 候选值：off, 0.4, 0.6, 0.8, 1.0（5个）
- 组合数：**5 组**

**总计：35 组**（相比全量笛卡尔积 180 组，缩减 80%）

### 评价指标

优先使用 **Calmar Ratio**（年化收益 / 最大回撤），因为该策略的过滤器本质上是用交易频率换取低回撤，Calmar Ratio 最能体现过滤器是否起到了正向作用。

### 参数高原验证

在选出最优参数后，检查其邻近参数的表现：
- 若 S\*=6，检查 S=5 和 S=7 的表现是否也接近
- 若 r\*=0.5，检查 r=0.4 和 r=0.6 的表现是否也接近
- 如果邻近参数表现骤降，说明该参数是"孤岛参数"，存在过拟合风险
