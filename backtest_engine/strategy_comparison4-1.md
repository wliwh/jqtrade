# ETF轮动策略研究(四之一)：动态ETF池轮动策略回测分析

## 概述

本文承接前文对[ETF轮动策略](https://www.joinquant.com/view/community/detail/8166da55303d8ab3f5095072771cb7f5)的参数调优与资产池适配分析，转而对另一技术路线——动态ETF池轮动策略——进行回测评估。与原版采用固定33只ETF候选池不同，动态池版采用**全市场动态扫描**：通过自动化规则在全市场ETF中筛选符合条件的标的，构建每日可交易的动态候选池。这一设计旨在突破固定池的视野局限，捕获更多潜在机会，但同时也引入了新的风险维度。

本次回测时间跨度为2023年初至2026年初的近3年周期，覆盖了市场从震荡到分化的完整阶段。回测结果显示，动态池版策略的年化收益区间约为20%至50%，最大回撤普遍在30%以上，部分参数组合下甚至触及40%红线。整体表现差强人意，与原版同周期动辄80%至100%的年化收益相比，存在显著差距。本文将详细剖析这一差距的成因，并探讨动态池策略的适用边界。


## 策略设计差异对比

动态池版策略与原版策略同属加权线性回归评分体系，继承了原策略的过滤风控框架，仅在标的池管理上走了截然不同的路线：

| 维度 | 原版固定池策略 | 动态池策略 | 备注 |
| :--- | :--- | :--- |:---|
| **池子构建** | 手动精选33只ETF，固定不变 | 全市场扫描后自动筛选 | |
| **上市时间过滤** | — | 上市不足60天则排除 | |
| **流动性门槛** | — | 日均成交额≥5000万元 | |
| **名称过滤** | — | 可指定过滤A股类、海外类、债权类ETF | |
| **指数去重** | — | 同一追踪指数仅保留成交额最大者 | 参考[引入指数跟踪](https://www.joinquant.com/view/community/detail/79f995d8d89bc6a3e67705308de93ada) | 
| **聚类优化** | — | 指数去重后，可选AP聚类进一步压缩池子 | 参考[ETF轮动池的构建](https://www.joinquant.com/view/community/detail/9d063c7c330e6d2378859f96aa5122fc) |
| **池子刷新频率** | — | 每N个交易日全量刷新，参数可配置 | |

从上述对比可以看出，动态池版策略的设计初衷是借助市场化的筛选机制，自动剔除流动性枯竭、上市过新、追踪同质指数的标的，从而在源头上提升候选池的质量。然而，实际回测结果表明，这一设计的表现并不理想。


## 几点简单的发现

1. 整体绩效偏低：近3年动态池版的回测结果普遍较差，年化收益一般在20%至50%区间，而最大回撤往往超过30%。与原版固定池策略动辄80%至100%的年化收益相比，差距显著。
2. 池子类型影响显著：如果动态ETF池去除A股标的，仅保留海外与商品ETF，收益有一定提升，最大回撤也有所降低。但即便如此，仍无法接近原版固定池策略的表现，说明问题的根源不完全在于A股标的的干扰。
3. 刷新频率至关重要：池子刷新频率参数对回测结果影响较大，将刷新周期从默认的20个交易日缩短至5至10个交易日能够显著改善绩效。过长的刷新周期会导致"过时"标的持续滞留在候选池中，增加选到"强弩之末"的风险。
4. AP聚类效果不确定：是否开启AP聚类对收益的影响并不稳定，在不同参数配置下可能产生正向或负向效果，尚未形成明确的优化方向。
