# ETF轮动策略研究(二)：标的池构建对策略绩效的决定性影响

## 引言

在第一篇报告中，我们验证了不同策略逻辑对收益和交易时间的敏感性。然而，一个更根本的问题往往被忽视：**策略的成功究竟源于优秀的择时逻辑，还是仅仅因为它“碰巧”坐在了最强的风口上？**

本篇报告将聚焦于**“ETF标的池 (Asset Pool)”**，探讨不同类型的资产组合（如纯A股、全球配置、行业轮动）如何从根本上决定动量策略的上限与下限。

### 1.2 评估方法论

为了更精确地衡量策略在不同标的池中的有效性，我们采用由宏观到微观的4层评估体系：

1.  **滚动收益对比**
    *   **方法**: 计算策略与池内所有ETF的 **N日滚动收益率**。
    *   **目的**: 观察策略的收益水平在池子中处于什么分位？是长期维持在前25% (Alpha)，还是随波逐流 (Beta)？

2.  **持仓周期归因**
    *   **方法**: 将回测分割为若干个“持仓不变阶段”。在每个阶段内，比较**策略持仓资产的收益** vs **池内其他资产的收益**。
    *   **目的**: 能够直观判断每一次选择是否正确。例如：策略持有黄金的20天里，黄金是否真的是池子里表现最好的？如果策略选了黄金但纳指涨得更好，说明选股逻辑在当前池子中失效。

3.  **相对强弱曲线**
    *   **方法**: 绘制双图：上方显示 `策略净值 / 标的池等权指数` 的比值曲线；下方显示 `策略净值` 与 `标的池等权净值` 的绝对走势对比。
    *   **目的**: 曲线向上代表有持续Alpha，曲线走平代表只有Beta，曲线向下代表跑输平均。这是判断“主动管理是否创造价值”最直观的图表。

4.  **换仓效果分析**
    *   **方法**: 追踪每一次策略发生换仓操作后，在持有新资产的期间内，新旧资产的收益差。
    *   **目的**: 衡量“瞎折腾”还是“神操作”。如果大部分换仓后的 Alpha 为正，说明策略具备优秀的短线择时能力（卖飞的情况少，买对的情况多）。

## 三、深度归因：Beta 还是 Alpha？

### 3.1 策略收益 vs 标的池平均收益
分析策略是否跑赢了标的池的等权持有收益。如果跑不赢，说明动量因子在该池子中失效（可能是因为同涨同跌，相关性太高）。

### 3.2 策略 vs 最佳单品 (Best Asset)
分析策略是否成功选出了池子中表现最好的那个资产。例如，在全球池中，策略是否能在美股牛市时一直以此为主？

## 四、相关性陷阱 (Correlation Trap)

*   **问题**: 当标的池内资产相关性过高时（如全是A股行业），动量策略往往失效。
*   **数据验证**: 计算各标的池的平均相关系数，并与其夏普比率进行回归分析。

## 五、研究结论与建议

### 5.1 什么样的池子适合动量策略？
*   **低相关性**: 资产之间最好是“你方唱罢我登场”，而不是“泥沙俱下”。
*   **长期正期望**: 池子里必须有“能涨”的资产。动量策略无法在垃圾堆里找金子。

### 5.2 针对不同池子的策略优化建议
*   对于高波动的行业池，建议增加风控阈值。
*   对于低波动的宽基池，建议放宽切换灵敏度。

基于上述分析，如果想让该动量策略继续保持“神级”表现，纳入池子中的ETF必须具备以下三个核心特征。这不仅仅是选品的标准，更是动量策略生效的物理公理。

1. 自身的长趋势性（Long-term Momentum）
要求：资产本身必须具有“一旦涨起来，就能持续很久”的特性。
原因：动量策略是迟钝的（Right-side Trading），它需要时间来确认趋势。如果一个ETF经常“涨三天跌两天”（如A股大部分行业板块），策略刚发出买入信号，行情就结束了，导致左右挨打。
正面例子：纳指100（长牛，一涨就是半年）、黄金（避险周期长）、创业板（弹性大，牛市不回头）。
反面例子：证券ETF（著名的“渣男”，脉冲式行情，极难捕捉）、传媒/基建（经常一日游）。
2. 彼此的低相关性甚至负相关性（Low Correlation）
要求：池子里的资产不能“同呼吸共命运”。最好是一个在跌的时候，另一个正在涨。
原因：动量策略的本质是**“去弱留强”**。如果大家一起强（全面牛市）或者一起弱（全面股灾），策略就失去了选择的意义：
全面牛市：策略频繁切换属于“瞎折腾”，跑不赢满仓持有。
全面熊市：策略无处可逃，只能硬扛亏损。
黄金标准：中美互补（纳指vs沪深300）、股债互补（30年国债vs股票）、股商互补（黄金vs股票）。必须引入这种跨市场、跨品种的资产。
3. 高波动但不高噪音（High Volatility, Low Noise）
要求：需要有足够的波动率来提供价差，但这种波动必须是有方向的（趋势），而不是无序的震荡（噪音）。
原因：
如银行ETF：波动率太低，即便选对了，扣掉手续费也赚不到什么超额收益。
如某些小盘题材ETF：波动率极高，但全是杂波，从数学上讲，动量因子在均值回归（Mean Reversion）的市场中是失效的。
理想标的：像纳指100那样，虽然每天都有波动，但拉长看斜率向上非常稳定；或者像原油，虽然波动剧烈，但趋势一旦形成就非常猛烈。


判定一个指数或ETF是否具有“长趋势性”（Long-term Trend / Trending Property），而不仅仅是随机漫步或均值回归，在量化金融中有一套成熟的数学和统计学评价体系。

不要只凭感觉（看K线图觉得在涨），要用数据说话。以下是几个核心指标，你可以直接用来筛选你的ETF池：

1. 赫斯特指数 (Hurst Exponent) —— 核心判据
这是衡量时间序列长期记忆性的最权威指标。

计算方法：基于重标极差分析（R/S Analysis）。
判读标准：($H$ 为赫斯特指数)
$0.5 < H < 1$：具有趋势性（持久性）。过去涨，未来大概率继续涨。$H$ 越接近 1，趋势越强。（这是我们要找的）
$H = 0.5$：随机漫步（布朗运动）。过去对未来无影响，纯噪音。
$0 < H < 0.5$：均值回归（反持续性）。涨多了会跌，跌多了会涨。（动量策略的死穴）
实操经验：纳指100的 $H$ 值通常在 0.65 左右（强趋势）；而A股震荡市期间的 $H$ 值往往在 0.45-0.55 之间（随机或回归）。

2. 动量持续性 (Momentum Persistence)
统计历史上“连续上涨/下跌 N 个月”后的概率分布。

计算指标：自相关系数（Autocorrelation, ACF）。
方法：计算月度收益率的滞后 $k$ 期自相关性（如 Lag-1, Lag-3, Lag-6）。
判定：
如果 Lag-1 和 Lag-3 的自相关系数显著为正，说明该资产具有动量效应。
如果系数接近 0 或为负，说明是震荡市。
3. VR 指数 (Volatility Ratio) —— 趋势/噪音比
这与技术指标里的VR不同，这里指“长期波动率 vs 短期波动率”的比值。

公式：$VR = \frac{\sigma_{Long}^2}{N \times \sigma_{Short}^2}$
例如：年化方差 / (12 × 月化方差)
判定：
如果 $VR > 1$，说明序列存在长期趋势扩张（趋势的力量大于随机波动的力量）。
如果 $VR < 1$，说明存在均值回归（短期噪音抵消了长期趋势）。
4. 稳健收益占比 (R-Squared of Log Price)
简单粗暴但有效：把价格取对数，做一个线性回归。

指标：线性回归的 $R^2$（拟合优度）。
意义：衡量价格走势是否像这种直线一样稳定向上。
判定：
$R^2 > 0.9$：超级稳健的长牛（如以前的白酒、美股）。这种是动量策略的最爱，因为回撤少，信号稳。
$R^2$ 低：说明走势上蹿下跳，即使最后涨了，中间也被洗出去无数次。
总结：一份量化体检表
下次你想加入一个新的ETF（比如越南市场、比特币、或者豆粕）时，先跑一下这个体检：

指标	必须满足的阈值	理想值	含义
Hurst指数	> 0.55	> 0.65	只要涨起来就停不下来
自相关系数 (Lag-1)	> 0	> 0.1	这个月涨，下个月还容易涨
对数线性回归 $R^2$	> 0.7	> 0.9	走势平滑，不容易被洗盘
年化波动率	> 15%	20%-30%	波动够大，才有价差可赚
结论：大部分A股行业ETF（证券、军工等）的 Hurst 指数都在 0.5 附近徘徊，甚至低于 0.5（经常一日游）。而纳指、黄金、比特币等资产，其 Hurst 指数长期显著大于 0.5。这就是为什么动量策略在A股难做，在美股好做的数学本质。
