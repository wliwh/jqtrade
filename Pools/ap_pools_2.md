# 2025-2026年ETF版图演变：铁打的核心，流水的边界

上一篇我们介绍了如何利用聚类算法构建ETF池，旨在解决传统人工建池中的“幸存者偏差”与“高相关性”问题，从而获得一个“低冗余、高代表性”的精英轮动池。本文将在此基础上，通过聚类追踪，进一步剖析2025-2026年间ETF市场版图的动态演变。

我们采用默认参数，对最近12期（2025/02-2026/01）的月度数据进行了连续聚类分析。通过追踪同一ETF在不同时期所属簇的变化，我们定义了**簇存活率**这一指标（即前一期某簇中的ETF在下一期维持在同一簇内的比例），并通过该指标量化市场风格的轮动特征。

## 1. 稳定性分析
*   **簇存活率**：70-80%。这意味着每月约有20-30%的标的发生类别迁移，量化验证了市场风格轮动的活跃性。
*   **核心资产**：尽管轮动剧烈，仍有20余个“核心家族”贯穿全周期始终存在，构成了市场的稳定基石。

## 2. 主流代表性ETF

在全年的演化中，出现频率最高、最具代表性的ETF包括：

1. 国内宽基：沪深300、中证500、创业板、科创板、科创100
2. 风格ETF：中证红利、红利低波、香港红利
3. 行业ETF：军工、光伏、证券、消费、人工智能、游戏、芯片、医疗、中概互联、香港创新药
4. 跨境ETF：纳指、德国、沙特、黄金、标普油气、豆粕

## 3. 市场洞察

1. 核心资产的演变
   沪深300与中证A500长期同属一个强相关簇。但在 **2025年6-7月** 和 **2026年1月**，二者出现了显著的分化，A500簇从300ETF簇中独立出来，显示出大盘核心资产内部的结构性分歧。

2. 中小盘的高度同质化
   *   “泛中小盘”集合：中证500、中证1000（如512100）和国证2000（如563300）长期被归类在同一个核心簇中。这表明过去一年A股的市值因子效应高度统一，单纯的市值下沉（500->1000->2000）已难提供独立的差异化走势。
   *   “蓄水池”效应：中证500表现出显著的**成长股“公共蓄水池”**特征。当机器人、金融科技等特定赛道产生强Beta时，相关成分股会从500中吸离归入主题簇；而当主题退潮，这些资产又回归平庸，重新融入中证500的宽基波动中。

3. 科技风格的裂变
    *   创业板：偶尔和光伏组、人工智能组发生流动。
    *   科创50的提纯：2025年10月，**半导体设备**从**科创50**中独立分化出来，显示市场对硬科技的交易颗粒度变细。
    *   科创100的独立：2025年6月，**科创100**脱离了**中证500/中小盘**的引力场，形成独立簇，这意味着科创板的小盘股走出了独立于主要板指的小票行情。

4. 红利策略的“三位一体”
   市场清晰保留了**中证红利、红利低波、香港红利**三条主线。值得注意的是，2025年6月后，该簇的代表从**银行ETF**变为**红利低波**。这说明银行板块在去年年中完成了属性切换，被市场彻底当成低波红利资产进行交易。

5. 跨境与商品资产
   *   境外资产：**纳指ETF**是当之无愧的境外领头羊，常年带动标普消费、东南亚科技等资产。相比之下，**日经、标普生物、德国、沙特**则表现出更强的独立性。
   *   商品资产：**黄金、标普油气、豆粕**走势独立，而**能源化工、有色**则经常与股票型有色ETF发生联动。
