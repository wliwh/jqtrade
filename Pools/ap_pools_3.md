# 2025-2026年ETF版图演变（番外篇）：引入指数跟踪，优化聚类精度

在上一篇关于[2025-2026年ETF版图演变](https://www.joinquant.com/view/community/detail/2ded392eddf7bddd0b3706922180488f)的分析中，我们探讨了核心资产的稳定性。春节期间，在翻阅 JoinQuant 的 API 文档时，我偶然发现可以通过 [etf跟踪指数信息](https://www.joinquant.com/help/api/help#fund:%E8%8E%B7%E5%8F%96etf%E8%B7%9F%E8%B8%AA%E6%8C%87%E6%95%B0%E4%BF%A1%E6%81%AF) 接口获取 ETF 对应的底层指数信息。

这一发现为我们的 ETF 聚类逻辑提供了极大的优化空间。例如，沪深300指数下挂钩了众多由不同基金公司发行的 ETF，它们的走势高度一致。通过识别并采用跟踪不同指数的 ETF 进行初步过滤，我们可以显著降低数据的冗余度，从而让聚类算法更聚焦于跨指数、跨风格的本质差异，进一步提升分析的精度。

### 指数过滤逻辑升级

![img](/asserts/query_index.png)

根据接口返回表格中的 `traced_index_name` 和 `traced_index_code` 两个字段，我们可以精准识别 ETF 所跟踪的指数。针对国内 ETF 市场及其多样化的跟踪指数品种（股票、债券、商品、QDII 等），我们的处理策略如下：

- Code 字段非空：优先使用 `traced_index_code` 作为合并的唯一键值，如图中绿色框中的ETF。
- Code 为空但 Name 非空：退而求其次使用 `traced_index_name` 作为键值，如图中蓝色框中的ETF。*注：考虑到可能存在名称略微差异但底层指数相同的情况，这部分逻辑未来仍有精进空间。*
- 两者均为空：目前观察到仅出现在商品黄金类的 ETF 中（图中橘黄色框中的），对此类特殊标的采用硬编码方式进行兜底处理。*后期可优化*

通过上述逻辑，我们提取出每个独立的指数及其对应的 ETF 集合。随后，在每个集合内部保留**上市时间最早**的 ETF（若上市时间相同，则保留代码编号最大者）作为该指数的代表，从而完成预过滤。

### 市场结构新洞察

除了算法与过滤逻辑的升级，我们在跑完最新的 12 期底层聚类数据后，也获得了关于市场核心主线演变的确凿结论：

1. **核心资产**
   聚类结果显示，各类核心宽基指数（如 `沪深300`、`中证A50`、`A500`，乃至 `上证50`、`上证180`、`深证100` 等）长期表现出极强的同质化。特别是 `A500`，自其纳入追踪以来，始终稳稳落在 `300ETF` 所在的簇中，从未发生分离。这意味着在月频轮动的视角下，这些大盘核心资产的 Beta 走势已被市场无差别且充分地定价。
2. **中小盘**
   * 年内绝大多数时期，`中证500`、`中证1000`、`国证2000` 甚至 `转债ETF`，都被算法牢牢地绑定在同一个泛小盘核心簇（通常以 `1000ETF` 为代表）中。
   * 同时，在这个核心中小盘簇周围，其他各细分主题 ETF 的流进流出相当频繁，这反映了中小盘极强的资金博弈特征与轮动活跃度。
3. **科技板块**
   * `光伏ETF` 在一年内辗转于 `创业板` -> `1000ETF` -> `化工` -> `电池` -> `电网设备` 等数个不同的簇之间。这反映它已失去超级赛道的独立定价权，退化为受制造周期或小盘 Beta 驱动的普通行业资产。
   * `新能源` 主题始终与 `创业板` 绑定在同一个簇中。考虑到新能源相关标的在创业板指中的极高权重，这意味着它们的核心走势已经高度共振，“买新能源往往等同于买创业板Beta”。
   * 大方向上形成了一硬一软两个稳定的科技簇。硬科技以 `芯片` 簇（含 `科创50`、`科创100`、`半导体设备`等）为代表，软科技则以 `5GETF/计算机` 簇（含 `人工智能`、`消费电子`、`云计算`等）为主导。两者构成了当前科技成长风格的主轴；同时，软科技簇与 `创业板` 簇的关联度极高，其内部成员经常相互渗透和流动。
4. **红利策略**
   * 2025年中旬，国内的 `红利` 与 `红利低波` 簇发生了实质性分离。分离后，`银行ETF` 普遍沉淀在更加注重确定性的 `红利低波` 簇中，而剩余的 `红利` 簇则由各类红利50/100宽基及 `电力ETF` 等组成。
   * `香港红利` 依然保持着极高的结构性独立。其内部成分高度稳定，通常牢牢锁定 `H股金融`、`央企` 等高确定性红利资产。
5. **跨境资产**
   * `日经225`、`德国` 等国别资产，在缺乏本国强劲独立宏观逻辑的时期，极易被统一吸卷入以 `标普500` 为核心的全球大簇中；只有在特定的政策窗口期才会脱钩并短暂自成一派。相比之下，`沙特` 与 `标普生物` 等细微跨境资产则表现出了极强的特立独行。
   * `黄金股ETF` 常年在 `黄金ETF`（避险属性）与 `有色ETF`（顺周期属性）之间游移，生动刻画了市场在交易“避险情绪”和“通胀预期”之间的横跳。而底层的纯商品资产（如 `黄金`、`豆粕`、`油气`）始终保持着极低的跨资产相关性，充当着整个配置池中最优秀的独立底仓资产。

除了核心过滤逻辑的升级，本次迭代还包含以下细节优化：
